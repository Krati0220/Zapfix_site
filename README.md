# ZapFix — Need a Fix? Zap It.

India's AI-powered home services platform. Upload a photo or video of your
appliance issue, get instant AI-powered diagnosis, transparent pricing, and a
verified professional at your door.

This repo contains the **landing website** for ZapFix — a production-grade,
mobile-first marketing site built with Next.js 14, Tailwind CSS, and Framer
Motion.

---

## Tech stack

- [Next.js 14](https://nextjs.org/) (App Router) + TypeScript
- [Tailwind CSS](https://tailwindcss.com/) with a custom navy / electric-yellow palette
- [Framer Motion](https://www.framer.com/motion/) for cinematic transitions
- [Lucide](https://lucide.dev/) icons
- Inter + Space Grotesk via `next/font`

## Sections

1. **Hero** — cinematic AI-first hero with animated phone mockup and floating diagnosis tags
2. **Problems** — the broken state of home repairs in India today
3. **How it works** — split-tab flow for customers and professionals
4. **AI Diagnosis** — live wireframe scanning UI with confidence-scored results
5. **Verified Professionals** — trust pillars + ZapPro profile cards
6. **Features** — 10 core platform features
7. **Service Categories** — AC, plumbing, electrical, painting and more
8. **App Screenshots** — 5 in-app screen mockups
9. **Download** — APK download CTA with floating phones (no Play Store / App Store)
10. **Vision** — cinematic mission statement
11. **Footer** — links, socials, big wordmark

## Local development

```bash
npm install
npm run dev
```

Then open <http://localhost:3000>.

## Production build

```bash
npm run build
npm start
```

## Deploy

This project is configured to deploy on [Vercel](https://vercel.com/) — push to
`main`, import the repo into Vercel, and it goes live with zero config.
